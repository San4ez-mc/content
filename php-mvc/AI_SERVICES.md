# 🤖 Підключені AI сервіси для генерації зображень

## Активні безкоштовні сервіси

### 1. 🎨 Pollinations.ai
- **Статус**: ✅ Повністю безкоштовний
- **Модель**: Flux (швидка генерація)
- **Обмеження**: Немає
- **Особливості**: 
  - Найшвидший сервіс
  - Висока якість генерації
  - Підтримка різних стилів
  - Параметри: width, height, model, nologo, enhance
- **URL**: https://pollinations.ai

### 2. 🧠 DeepAI
- **Статус**: ✅ Безкоштовний tier
- **Обмеження**: 150 запитів на місяць
- **API**: Text2Image
- **Особливості**:
  - Різні стилі генерації
  - Швидка обробка
  - Без обов'язкового API ключа (публічний ендпоінт)
- **URL**: https://deepai.org

### 3. 🤗 Hugging Face Inference API
- **Статус**: ✅ Безкоштовні inference запити
- **Модель**: Stable Diffusion 2.1
- **Обмеження**: Rate limiting при великій кількості запитів
- **Особливості**:
  - Професійна якість
  - Підтримка різних моделей
  - API без ключа для публічних моделей
- **URL**: https://huggingface.co/inference-api

### 4. 🎭 Craiyon (DALL-E mini)
- **Статус**: ✅ Безкоштовний
- **Модель**: Craiyon v3 (photo mode)
- **Обмеження**: Повільна генерація (60-120 секунд)
- **Особливості**:
  - Креативні результати
  - 3 версії моделі: art, drawing, photo
  - Negative prompts підтримуються
- **URL**: https://craiyon.com

### 5. 💻 Local Processing (Fallback)
- **Статус**: ✅ Завжди доступний
- **Технологія**: PHP GD Library
- **Особливості**:
  - Працює на вашому сервері
  - Моментальна обробка
  - Застосування фільтрів: brightness, contrast, colorize, smooth
  - Використовується якщо всі AI сервіси недоступні

## Логіка роботи

Система намагається згенерувати варіації через сервіси **по черзі**:

1. Спочатку Pollinations.ai (найшвидший)
2. Якщо не вдалось → DeepAI
3. Якщо не вдалось → Hugging Face
4. Якщо не вдалось → Craiyon
5. Якщо всі AI недоступні → Local Processing (fallback)

Кожна варіація генерується окремо, тому можуть використовуватись різні сервіси для різних варіацій.

## Кількість варіацій

- **За замовчуванням**: 5 варіацій на кожне оригінальне фото
- Кожна варіація має унікальний prompt:
  1. Різна поза
  2. Різний одяг
  3. Різне оточення
  4. Candid стиль
  5. Artistic портрет

💪 **Найнадійніший**: Local Processing (завжди працює)

## Troubleshooting

**Якщо генерація не працює:**
1. Перевірте інтернет-з'єднання на сервері
2. Переконайтесь що cURL та GD library встановлені
3. Перевірте SSL сертифікати (або вимкніть verify_peer)
4. Система автоматично використає Local Processing як fallback

**Якщо варіації низької якості:**
- Спробуйте згенерувати ще раз (може використати інший сервіс)
- Завантажте оригінал вищої якості
- Hugging Face та Pollinations.ai дають найкращі результати
