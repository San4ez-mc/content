# 📁 Папка для початкових зображень

## Розташування
`public/uploads/source_images/`

## Призначення
Ця папка призначена для зберігання початкових фотографій, на основі яких буде генеруватися контент для соціальних мереж.

## Як використовувати

### Спосіб 1: Через веб-інтерфейс (рекомендовано)
1. Увійдіть в систему
2. Перейдіть в меню **"Зображення"**
3. Натисніть кнопку **"Завантажити"**
4. Виберіть файл (JPG, PNG, GIF, WebP до 10MB)
5. Натисніть кнопку **"✨ Згенерувати копії"** для створення варіацій

### Спосіб 2: Пряме завантаження
Просто скопіюйте ваші фото в цю папку через файловий менеджер або FTP.

## Підтримувані формати
- JPG / JPEG
- PNG
- GIF
- WebP

## Обмеження
- Максимальний розмір файлу: **10 MB**
- Рекомендована якість: високе розширення для кращих результатів AI генерації

## Що відбувається після завантаження?
1. Файл зберігається в `source_images/`
2. При натисканні "Згенерувати копії" система створює варіації
3. Згенеровані варіації зберігаються в `generated_images/`
4. Варіації можна використовувати в контент плані

## Примітки
- Зображення в цій папці НЕ видаляються автоматично
- Для видалення використовуйте кнопку 🗑️ у веб-інтерфейсі
- Оригінальні файли залишаються незмінними при генерації варіацій
